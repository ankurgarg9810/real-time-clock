# Real-Time Clock

A simple real-time digital clock built using HTML, CSS, and JavaScript.  
The application displays the current system time and updates automatically every second.

## Features
- Live time display
- Updates every second
- Uses JavaScript Date object
- Automatic refresh with timer function
- Clean and minimal UI

## Tech Stack
- HTML
- CSS
- JavaScript
